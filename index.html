Recommended flow
How to use this suite
1
Choose your task
Start with Task A (individual) before Task B (group). Weaker or moderate students should always practise Task A first.
2
Follow the steps on-screen
Each task has 4 guided steps. Complete them in order — selecting your set, reading the question, using the timer, then checking feedback.
3
Practise, then improve one thing
Don't try to fix everything at once. Read your feedback and improve just one sentence or phrase before your next attempt.
4
Repeat with a new set
Speed and confidence come from repetition. Practise the same structure across different topics to build fluency.
Speaking Test Flow
How the MUET Speaking Test Is Conducted
🧭
One topic, two tasks
Task A and Task B use the same set or topic. First, each candidate speaks individually in Task A. After that, the same candidates continue with Task B by discussing the same topic together through support, agreement, disagreement, and response.
Step-by-Step Test Conduct
Use this page to explain the full speaking test to students
📝
1
Students receive one MUET topic set
All four candidates in the same group use the same booklet or prompt set. The topic in Task B comes from this same set.
2
Candidates prepare for Task A
Each candidate gets a different point under the same Task A situation. They have 2 minutes to prepare short notes.
3
Task A individual presentation begins
Candidate A, B, C, and D speak one by one. Each candidate presents for about 2 minutes using their own assigned point.
4
Candidates move to Task B using the same topic
After all individual presentations, the group continues with Task B. They do not change to a different topic. They discuss the same set in an interactive way.
5
Candidates prepare for Task B
The group gets 3 minutes to prepare. Students review the Task B question and choose ideas they want to support during discussion.
6
Task B group discussion starts
All four candidates discuss together for 10 minutes. They support ideas, agree, disagree politely, ask questions, respond to each other, and work towards one best answer.
7
The group reaches a conclusion
Before the discussion ends, candidates should help the group agree on the most suitable or most effective point.

Step 4 — Read your feedback and improve one thing
Look at your strengths first — that's what you're already doing well. Then pick just one area to improve. Practise that specific thing before clicking New Set to try again.
Quick Improvement Guide
1
If Band 1–2: Focus on completing one full sentence
Just say your point + one reason. Two clear sentences is better than five confused ones.
2
If Band 3–4: Add a specific example
"For example, in Malaysia..." — any real-life context. This moves you from Band 3 to Band 4.
3
If Band 4–5: Use more connecting words
"Furthermore", "This is mainly because", "As a result" — these improve coherence.
4
If Band 5–6: Upgrade vocabulary
Replace "good" → "beneficial", "bad" → "detrimental", "think" → "perceive".
